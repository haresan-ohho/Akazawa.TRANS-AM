概要
　ペンギンを操って曲芸をするゲームです。
　泳ぐ速度、ジャンプする高度、飛距離、曲芸時間、水飛沫の大きさなどから得点が加算されていきます。
　

動作条件
　Android 2.2以上
　MB以上の空き容量が必要

ファイル構成

インストール

アンインストール
　設定→アプリケーションの管理→ペンギンゲーム(仮)→アンインストール

使用法
 ゲームを起動するとメニュー画面が出ます
 メニュー画面からHow toを選択するとゲームの操作の説明やスコア解説が見れます
 　How to画面では次の画面に進む際、画面にタッチしスライドさせることで次の画面を見る事が出来ます
 ゲームを終了させる時はメニュー画面の終了を押すか、Android端末のボタンで終了させる事が出来ます
